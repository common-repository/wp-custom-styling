Now you can customize your heading, blockquotes, boxes designs in any of your wordpress themes using WP Custom Styling Wordpress plugin.

== Description ==

This Plugin allow you to add pre designs in any of your post/pages by just one click.You can change color of text & choose different designs by preview.

== Installation ==

1. Upload `wp-custom-styling.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now you can use WP Custom Styling

== Frequently asked questions ==

* How to use WP Custom Styling ?
Ans- You can use this plugin on any page or post using icon on Wp Editor.Just select test which you want to customize and click on icon to design this text.
